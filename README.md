## Installation

First, you need install the package first before ready to install.

```bash
pkg install wget
```

You need to install XFCE4 Desktop before install Wine Hangover.

```bash
wget https://github.com/LukeYT-TermuxForAndroid/XFCE4-Setup-Wine-Hangover-for-Termux/raw/refs/heads/main/setup/xfce4_setup.sh && chmod +x xfce4_setup.sh && ./xfce4_setup.sh
```

Then, you need install Wine Hangover to run Windows applications.

```bash
wget https://github.com/LukeYT-TermuxForAndroid/XFCE4-Setup-Wine-Hangover-for-Termux/raw/refs/heads/main/setup/wine_hangover.sh && chmod +x wine_hangover.sh && ./wine_hangover.sh
```
